# 概要
書籍検索APIを使用して本のタイトルから書籍情報を検索するアプリです  

![スクリーンショット (58)](https://user-images.githubusercontent.com/69840682/150519523-c38ff2cd-b1ed-4e6a-a4b1-382d1a7d7d26.png)

## 環境構築方法/セットアップの仕方
ローカル環境での立ち上げになります  
以下のコマンドをお願いします  
clone  
$ git clone   
run  
$ npm install  
$ npm start  

### 技術スタック
・React  
・TypeScript  
・MATERIAL-UI  
・axios  

### 機能
・書籍の検索機能（本のタイトルによるキーワード検索）  
・書籍検索結果の表示
